# Abrupt
### Rust Package
##### **(Cryptographic & Mathematic)**

## Modules
#### Cryptographic Operations
1. **Hash Algorithms**
    - SHA-256
    - SHA-512
    - MD5
2. **Data encoding**
    - Base16 (Hexadecimal)
    - Base32
    - Base64
#### Mathematical Utilities
1. **Array Computation**
    - Summation
    - Product Calculation
    - Average Determination
    - Minimum and Maximum Value
2. **Number Theory**
    - Factorial
    - Greatest Common Divisor (GCD)
    - Least Common Multiple (LCM)
    - Primality Testing
2. **Fundamental Mathematical Operations**
    - Exponentiation
    - Absolute Value
    - Value Clamping

## Installation

**Cargo Command:**
```sh
cargo add abrupt
```

**Cargo.toml:**
```toml
[dependencies]
abrupt = "0.1.0"
```
